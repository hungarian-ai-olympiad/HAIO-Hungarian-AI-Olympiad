# HAIO 2025 | Nyári Online Válogató

Platform: Kaggle

## Egyetemi Felvételi (ML)

Gépi tanulási modell fejlesztése, amely egyetemi felvételi eredményeket jósol korábbi évek adatai alapján.

**Értékelés:** ROC-AUC

**Továbbjutás:** A baseline feletti pontszám szükséges.

- [Kaggle verseny](https://www.kaggle.com/competitions/magyar-mi-diakolimpia-online-valogato-2025)
